# Roof Reservation — Real Floor Map Patch

این نسخه، نقشه فرضی قبلی را با نقشه واقعی و سه‌سالن Roof جایگزین می‌کند:

- سالن پنجره
- سالن وسط
- روف گاردن

نقشه با SVG مستقل، میزهای داده‌محور، زوم و جابه‌جایی، حالت‌های رزرو و ویرایشگر میز در پنل ادمین ساخته شده است.

## روش جایگزینی

محتویات ZIP را دقیقاً روی ریشه پروژه `roof-reservation` کپی و فایل‌های هم‌نام را Replace کن.

ساختار ZIP به همان شکل پروژه است:

```text
prisma/seed.js
public/index.html
public/admin.html
public/css/style.css
public/js/reservation.js
public/js/admin.js
public/js/map-renderer.js
public/js/admin-map-editor.js
public/assets/maps/roof-base.svg
public/assets/maps/roof-map.json
README.md
```

## اعمال اطلاعات میزهای جدید

بعد از جایگزینی فایل‌ها، در ریشه پروژه اجرا کن:

```bash
npm run db:seed
```

Seed جدید:

- ۴۶ میز را در سه سالن ایجاد یا به‌روزرسانی می‌کند.
- شماره‌های نمایشی فعلاً موقت هستند و از پنل ادمین قابل تغییرند.
- ظرفیت میزها بر اساس صندلی‌ها و مبل‌های پلان اولیه تنظیم شده است.
- اتصال‌های مجاز برای ترکیب حداکثر دو میز را می‌سازد.
- میزهای فرضی قدیمی `T1` تا `T10` را، در صورت نداشتن رزرو، حذف می‌کند.
- اگر میز قدیمی رزرو ثبت‌شده داشته باشد، آن را حذف نمی‌کند و فقط غیرفعال نگه می‌دارد.

سپس پروژه را اجرا کن:

```bash
npm run dev
```

## مدیریت میزها

در پنل ادمین وارد بخش «مدیریت میزها» شو. مالک و مدیر می‌توانند:

- میز را روی نقشه جابه‌جا کنند.
- شماره نمایشی را تغییر دهند.
- ظرفیت، حداقل و حداکثر نفر را ویرایش کنند.
- شکل، اندازه و چرخش میز را تغییر دهند.
- میز را فعال یا غیرفعال کنند.
- اتصال مجاز بین دو میز را اضافه یا حذف کنند.

نقش پذیرش فقط نقشه و اطلاعات میزها را مشاهده می‌کند.

## معماری نقشه

```text
roof-base.svg
```

فقط معماری و دکور ثابت را نگه می‌دارد: دیوارها، پنجره، مبل سرتاسری، پله، صندوق، کافه‌بار، فضای خدماتی، سرویس، درها، فضای سبز و نشانه‌های اصلی.

```text
roof-map.json
```

محدوده سالن‌ها، نمای زوم، تنظیمات ظاهری صندلی‌ها و هندسه اولیه میزها را نگه می‌دارد.

```text
map-renderer.js
```

میزها، صندلی‌ها، وضعیت آزاد یا رزروشده، برچسب ساعت، انتخاب ترکیبی، زوم، پن و تمرکز روی سالن‌ها را داخل SVG می‌سازد.

```text
PostgreSQL
```

منبع اصلی شماره میز، ظرفیت، زون، مختصات، اندازه، چرخش، فعال‌بودن و اتصال میزهاست.

## نکته

برای اعمال این Patch نیازی به Migration جدید یا تغییر `schema.prisma` نیست. مدل فعلی `CafeTable` تمام فیلدهای موردنیاز نقشه را از قبل دارد.
